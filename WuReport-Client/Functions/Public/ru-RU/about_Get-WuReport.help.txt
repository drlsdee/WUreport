TOPIC
    about_Get-WuReport
SYNOPSIS
    Функция собирает отчет о Центре обновления Windows на локальном компьютере и при необходимости запускает отложенные действия.
DESCRIPTION
    Функция собирает отчет о Центре обновления Windows на локальном компьютере. Отчет должен включать такие данные, как:
    - WSUS URI;
    - метка времени последней установки обновлений;
    - отметка времени последнего обнаружения ожидающих выполнения действий (таких как перезагрузка или установка обновлений);
    - коллекции успешных, неудачных, прерванных и ожидающих обновлений Windows;
    - список интерактивных пользователей;
    - и так далее.
    Функция также отображает сообщение для всех интерактивных пользователей, если было обнаружено хотя бы одно из действий, ожидающих выполнения.
    Если локальный компьютер ожидает перезагрузки или установки обновлений более определенного количества рабочих дней, функция выполнит все действия немедленно.
    Все отчеты хранятся в виде файлов JSON в папке, определенной в параметрах. Если папка не определена, путь к папке по умолчанию - "$env:ProgramData\PSCustomData\<ModuleName>\"
EXAMPLE
    PS C:\> Get-WuReport -ServiceType MicrosoftUpdate -Path C:\reports\ -PSCustomDataFolder Data -Weekends @('Sunday', 'Saturday') -DelayMax 3 -TimeToDisplay 60 -RebootTimeout 180
    В качестве источника обновления будет установлен Центр обновления Майкрософт (Microsoft Update).
    Папка для хранения отчетов - "C:\reports". Параметр "PSCustomDataFolder" в этом случае будет проигнорирован.
    Суббота и воскресенье - выходные.
    Максимальная задержка отложенных действий составляет 3 рабочих дня.
    Сообщение должно отображаться у пользователей в течение 60 секунд.
    Тайм-аут перед перезагрузкой (при необходимости) будет установлен на 180 секунд.
INPUTS
    [System.String]
    [System.String[]]
    [System.Int32]
OUTPUTS
    $null
NOTES
    Не забывайте о "second hop problem", когда вы запускаете эту функцию из удаленного сеанса PowerShell и ожидаете, что данные отчета будут сохранены на удаленном файловом ресурсе.
